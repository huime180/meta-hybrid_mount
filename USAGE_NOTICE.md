# Hybrid Mount 使用须知 / User Notice

## 中文

在安装与使用 Hybrid Mount 前，请确认你已知悉以下事项：

1. 本模块会修改系统挂载行为，可能与设备环境或其他模块产生兼容性问题。
2. 若出现异常（如开机卡顿、应用异常、分区挂载失败），请先禁用或卸载本模块后重启。
3. 请仅在理解风险、具备基础排障能力的前提下继续使用。
4. 继续安装或使用即表示你已阅读并同意自行承担相关风险。
5. 如需反馈问题，请将管理器中的 bugreport 发送到 GitHub Issues，或在 Telegram 群组 `@hybridmountchat` 反馈。

## English

Before installing or using Hybrid Mount, please read and acknowledge:

1. This module changes system mount behavior and may conflict with your device setup or other modules.
2. If issues occur (boot problems, app failures, partition mount errors), disable or uninstall this module and reboot first.
3. Proceed only if you understand the risks and can perform basic troubleshooting.
4. Continuing installation or usage means you have read this notice and accept responsibility for potential risks.
5. For issue reporting, send the manager bugreport to GitHub Issues, or report it in the Telegram group `@hybridmountchat`.
